Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

POST api/residents

Example request:
curl --request POST \
    "http://localhost/api/residents" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "email=boehm.archibald@example.org"\
    --form "building_id=vitae"\
    --form "document=fkfxydfumvy"\
    --form "first_name=avutxtcunmsikfcffqkn"\
    --form "last_name=yx"\
    --form "phone=omcg"\
    --form "apartment=ljtc"\
    --form "avatar=@/tmp/phpMiiGNa" 
const url = new URL(
    "http://localhost/api/residents"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('email', 'boehm.archibald@example.org');
body.append('building_id', 'vitae');
body.append('document', 'fkfxydfumvy');
body.append('first_name', 'avutxtcunmsikfcffqkn');
body.append('last_name', 'yx');
body.append('phone', 'omcg');
body.append('apartment', 'ljtc');
body.append('avatar', document.querySelector('input[name="avatar"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());

POST api/residents

Headers

Content-Type      

Example: multipart/form-data

Accept      

Example: application/json

Body Parameters

email   string   

Must be a valid email address. Must not be greater than 255 characters. Example: boehm.archibald@example.org

building_id   string   

The id of an existing record in the buildings table. Example: vitae

document   string   

Must not be greater than 20 characters. Example: fkfxydfumvy

first_name   string   

Must not be greater than 50 characters. Example: avutxtcunmsikfcffqkn

last_name   string   

Must not be greater than 50 characters. Example: yx

phone   string   

Must not be greater than 20 characters. Example: omcg

avatar   file  optional  

Must be an image. Must not be greater than 2048 kilobytes. Example: /tmp/phpMiiGNa

apartment   string   

Must not be greater than 20 characters. Example: ljtc
